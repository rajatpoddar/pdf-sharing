# **App Name**: Weekly Projects Hub

## Core Features:

- PDF Upload: Admin uploads PDF files for each week, associating them with specific weekdays (Monday-Sunday).
- PDF Search: Users can search for PDF files by filename or week.
- PDF Filtering: Users can filter PDF files by week to find relevant documents.
- PDF Download: Users can directly download the PDF files.
- Mobile-Optimized PDF Display: Display the PDF name and week. Each PDF should be represented in a grid layout optimized for mobile viewing.
- AI Project Suggestion Tool: An AI-powered tool suggests relevant projects by learning user preferences through their past downloads.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey reliability and accessibility, suitable for a document-focused app.
- Background color: Light gray (#F0F0F0) to provide a clean and neutral backdrop for the content.
- Accent color: A soft green (#90EE90) for buttons and highlights, complementing the blue for a balanced feel.
- Body and headline font: 'PT Sans', a modern humanist sans-serif that provides a blend of clarity and approachability for comfortable reading on mobile devices.
- Use simple, clear icons for categories and actions to enhance usability on mobile devices.
- Maintain a clean and intuitive layout optimized for mobile viewing, with clear sections for search, filtering, and PDF display.
- Use subtle animations to enhance user experience, such as smooth transitions when filtering or downloading PDF files.
